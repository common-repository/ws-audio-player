====================
描述
====================
这个一个可在日志插入且附带下载功能的在线Flash播放器。


====================
安装
====================
1.解压缩ZIP文件到自己BLOG空间的wp-content/plugins下面。
2.进入WP后台激活WS Audio Player插件。
3.在WP后台的Option（操作）管理页面，找到WS Audio Plauer可进行外观的选择和配色。


====================
使用方法
====================
发布日志的时候，你可以在编辑器找到按钮来插入音乐或者在代码状态输入：[audio=音乐的地址[,音乐名称,download]]

(*)其中音乐名称和download（是否可下载）为可选择选项。 

举例说明： 插入一首名为“爱转角”，音乐的地址为http://icyleaf.com/music.mp3 
样式1： 只显示播放器
[audio=http://icyleaf.com/music.mp3] 

样式2： 只显示播放器以及音乐名称
[audio=http://icyleaf.com/music.mp3,爱转角] 

样式3： 只显示播放器以及下载按钮（注意，中间有两个逗号）
[audio=http://icyleaf.com/music.mp3,,download] 

样式4： 全部显示：播放器，音乐名称，下载按钮
[audio=http://icyleaf.com/music.mp3,爱转角,download] 


====================
注意事项
====================
1. 除了音乐名称外，其他符号均为英文半角状态。
2. 如果只准备显示播放器以及下载按钮，不要忘记中间为两个逗号。
